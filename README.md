# Robotics (ROB) - UPC

Este repositorio contiene las prácticas y el proyecto final de la asignatura de Robótica (ROB) de la Universitat Politècnica de Catalunya (UPC).

## Estructura del repositorio

- **Lab2**: Contiene paquetes de ROS 2 para la navegación básica y el análisis de datos de LiDAR con TurtleBot3.
- **Projecte**: Proyecto final de Navegación Autónoma. Incluye:
  - Navegación global con evasión de obstáculos.
  - Exploración del área (Passadís) procesando datos de LiDAR y SLAM para crear mapas.
  - Detección de una estación de carga (usando clustering de pilares).
  - Aparcamiento de precisión coordinado.

## Tecnologías principales
- **ROS 2** (Robot Operating System)
- **Python**
- **TurtleBot3** (Burger)
